This is a flight search terminal application. The User will be able to look up 2 cities, one for the origina and destination, then input 2 dates referring to the start and end of a range of dates. The Application will take this information and using the kiwi.com API will return all the possible flights for those parameters. The User will then be able to choose one of the flights and the application will return a Deep Link that will take the user straight to the booking page for the flight.
Notable features: Written in OOP, Consumes an API, Automated Testing, CI/CD Pipeline
